Detailed Description

This class is responsible for parsing the .txt file and extracting the data from it.

The files must follow the specified format:

ranges_.txt:

comment line

varName: lineNum1[+|-], lineNum2[+|-], ...

registers_.txt:

comment line

registers: N

algorithm variants: basic, splitting and spilling each with a numeric parameter

algorithm: basic
